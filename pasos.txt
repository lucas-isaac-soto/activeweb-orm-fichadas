1 Paso uno definicion de tablas de base de datos.

En la base de datos postgres, crear un esquema con el nombre de fichadas.

En el esquema de fichadas crear la tabla empleado,ciudad,sucursal y movimientos.

empleado tendra como atributos:
    id
    nombre 
    apellido
    documento
    id_ciudad           ->   tabla ciudad
    id_sucursal         ->   tabla sucursal
    fecha_nacimiento
    sueldo

ciudad tendra como atributos:
    id 
    nombre

sucursal tendra como atributos:
    id
    id_ciudad
    id_empleado_gerente

movimientos tendra como atributos
    id
    id_empleado
    fecha_hora_ingreso

crear pocos datos de prueba para consumir
